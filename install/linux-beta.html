TexText for Inkscape 1.0 RC1 on Linux (Ubuntu 18.04)¶

This guide explains how to install and use TexText for Inkscape 1.0 RC1 together with Inkscape 1.0 RC1 on a system on which also Inkscape 0.92.x is installed. Unfortunately, this approach works only with Ubuntu 18.04.

Preparation¶

  1. Backup your existing Inkscape extension directory

    cp -r ~/.config/inkscape/extensions ~/.config/inkscape/extensions.backup

  2. Remove the old TexText installation:

    rm -rf ~/.config/inkscape/extensions/textext
rm ~/.config/inkscape/extensions/textext.inx

Download and install the Inkscape 1.0 RC1 app image file¶

  1. Download Inkscape 1.0 RC1 build 09960d6 from Gitlab or use the following command in a terminal:

    wget -c https://gitlab.com/inkscape/inkscape/-/jobs/505628089/artifacts/raw/Inkscape-09960d6-x86_64.AppImage

  2. Change into the directory into which you downloaded the AppImage. Then, make it executable:

    chmod u+x Inkscape-09960d6-x86_64.AppImage

  3. Extract the AppImage:

    ./Inkscape-09960d6-x86_64.AppImage --appimage-extract

  4. Optional, if you are not in your home directory: Move the created directory squashfs-root into your home directory:

    mv squashfs-root ~

After this operation Inkscape 1.0 RC1 will reside in ~/squashfs-root/bin. You can test it by executing

~/squashfs-root/bin/inkscape &

Install GUI library¶

Install the Python bindings for the graphical user interface of TexText for Inkscape 1.0 RC1. You have two options: GTK3 (recommended) or Tkinter.

At first you need to discover the Python interpreter that is used by your Inkscape installation. Enter the following command in a terminal

python --version

Keep the returned major version number (Python 2 or Python 3) in mind for the following instructions:

Install Python GTK3 bindings (recommended)¶

If your Inkscape installation runs Python 2 you need the Python 2.x bindings for gobject-introspection libraries (python-gi), the GTK+ graphical user interface library (gir1.2-gtk-3.0) and the gir files for the GTK+ syntax highlighting widget (gir1.2-gtksource-3.0)

sudo apt-get install python-gi gir1.2-gtk-3.0 gir1.2-gtksource-3.0

If your Inkscape installation runs Python 3 you need the Python 3 version of the gobject-introspection. The rest remains the same:

sudo apt-get install python3-gi gir1.2-gtk-3.0 gir1.2-gtksource-3.0

Install Tkinter (not recommended)¶

Tkinter is functioning but has a limited interface compared to GTK version, so it’s not recommended. To use Tkinter install the Python tk package.

If your Inkscape installation runs Python 2:

sudo apt-get install python-tk

If your Inkscape installation runs Python 3:

sudo apt-get install python3-tk

Download and install TexText for Inkscape 1.0 RC1¶

  1. Download the most recent preview package from GitHub release page

  2. Extract the package and change into the created directory.

  3. Run setup.py with (Important!!) specification of the path to your Inkscape 1.0 RC1 executable from your terminal:

    python3 setup.py --inkscape-executable ~/squashfs-root/bin/inkscape

    Setup will inform you if some of the prerequisites needed by TexText for Inkscape 1.0 RC1 are missing. Install them.

    Important

    Compared to previous versions of TexText for Inkscape 0.92.x TexText for Inkscape 1.0 RC1 does not need any conversion utilities like ghostscript, pstoedit or pdfsvg.

Now you can launch Inkscape 1.0 RC1 by typing ~/squashfs-root/bin/inkscape & and work with TexText for Inkscape 1.0 RC1

Please report any issues! Thank you!

Switching back to Inkscape 0.92.x¶

mv ~/.config/inkscape/extensions ~/.config/inkscape/extensions.beta
cp -r ~/.config/inkscape/extensions.backup/ ~/.config/inkscape/extensions
